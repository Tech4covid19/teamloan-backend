# TeamLoan backend API
<img src="/src/main/resources/META-INF/resources/template-resources/mail-header-joined.jpg" width=100%>

This project is the [TeamLoan](https://www.teamloan.pt) backend API.

## Quarkus framework

This project uses Quarkus, the Supersonic Subatomic Java Framework.

If you want to learn more about Quarkus, please visit its website: https://quarkus.io/ .

## Requirements for development
* [JDK 11](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot)
* [Apache Maven 3.6.2+](https://maven.apache.org)

## Running the application in dev mode

You can run your application in dev mode that enables live coding using:
```
mvn quarkus:dev
```

You can then attach a debugger on the port 5005 (default) using your IDE.
